Detailed Description

template<class T>
class connection_hub::streams::LatestRingBuffer< T >

Ring buffer that overwrites old values and exposes the most recent one.

LatestRingBuffer stores up to a fixed number of elements but only exposes the most recently published value to consumers. Older values may be overwritten as new data is published.

This type is thread-safe for concurrent publishers and receivers.

TType of element stored in the buffer.
