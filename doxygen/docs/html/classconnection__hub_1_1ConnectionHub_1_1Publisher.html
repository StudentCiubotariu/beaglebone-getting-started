Detailed Description

template<typename MessageT>
class connection_hub::ConnectionHub< MessageT >::Publisher

Lightweight handle used to publish messages into a ConnectionHub stream.

A Publisher provides write-only access to an internal message queue. It does not own the queue; the lifetime of the queue must exceed the lifetime of the Publisher.

This class is intentionally small and cheap to copy or move.

Note
Thread-safety depends on the underlying MsgQueue implementation.
