Detailed Description

Core processing Logger for message logging with levels.

Logger is movable but not copyable.

enum Logger::Level : std::uint8_t
strong

Log severity levels.

Defines the verbosity/severity of log messages. The enum uses an explicit underlying type of uint8_t to keep storage compact and predictable.

Enumerator
OFF 

No logging.

INFO 

Informational messages.

WARN 

Warnings.

ERROR 

Recoverable errors.

FATAL 

Unrecoverable errors.
